# Kindle Weather Station (PW2)

Vibe-coding a Kindle weather display: I have a 4th Generation Kindle Touch (Model D01200) and asked ChatGPT: "I have an old kindle. Help me set up a weather station via jailbreak and Github actions".

Low-power weather dashboard for an old Kindle Paperwhite 2.

## Features
- Meteostat (no API keys)
- GitHub Actions auto-update
- E-ink optimized PNG
- Liebefeld, Switzerland

## Setup
1. Fork repo
2. Enable GitHub Pages (root)
3. Wait for action to run
